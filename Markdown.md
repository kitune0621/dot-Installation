#### Markdown記法入門

##01 Markdown記法とはなにか？

* 概要
 - 軽量マークアップテキスト  
 - プレインテキスト → HTML  
 # Title → \<h1>Title\</h1>
 
* 公式サイト  
http://daringfireball.net/projects/markdown/

* 知識
 - HTML
 - Sublime text
 
* 準備
 - sublime textからMarkupdownの文章をプレビューするためのパッケージをダウンロードする  
   ※パッケージ:https://packagecontrol.io/installation
